# Rollup-Web

## 介绍

Rollup-Web 是一个基于浏览器的打包工具。它可以使你在浏览器中运行 TS、Vue 文件，甚至是其他语言！Rollup-Web 使用 TS 进行项目的编写，并使用了很多高级特性帮助不同类型的文件在 JS 环境中运行，你只需要像使用 Rollup 一样，编写编译脚本即可，剩下的事情，Rollup-Web 会自己解决！

> Rollup-Web 与 Vite、Rollup、Webpack 等运行在 Nodejs 端的打包工具不同，它只关注于让开发者可以在浏览器上运行多种不同的项目文件，并不在意打包成果，但是他们的运行方式是类似的。
